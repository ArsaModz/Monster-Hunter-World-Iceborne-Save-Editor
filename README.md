A feature-rich WPF desktop application for editing Monster Hunter World: Iceborne save files on PC and PS4. Built with C# / .NET 4.7.2, it handles save decryption and re-encryption transparently using Blowfish (PC) and PS4 crypto, so you always work with clean, valid saves.
Features

🔐 PC & PS4 save support — automatic decryption/encryption with Steam ID editing for PC saves
🎒 Full inventory editing — Item Pouch, Ammo Pouch, Item Box, Ammo Box, Material Box, and Deco Box (up to 1,250+ slots), with searchable item lookup and bulk-add
⚔️ Equipment box — edit all 2,500 equipment slots including decorations, augments, advanced augments, awakened abilities, bowgun mods, and pendants
🐱 Palico support — Palico name, XP, gadget levels, and full Palico equipment box editing
📋 Loadout management — view and edit saved equipment loadouts
🧑 Character editor — hunter name, gender, HR/MR, Zenny, Research Points, Steamfuel, play time, and decoration unlock flag
🗡️ Equipped gear editor — directly edit currently worn weapon, armor, charm, tools, and kinsect
🔧 Mantle upgrades — toggle all 20 mantle/booster upgrades
💾 PC ↔ PS4 save conversion
